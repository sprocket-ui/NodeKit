# @sprocketui-types/tabs

Framework-agnostic types for Sprocket UI Tabs component.

## Installation

```bash
pnpm add @sprocketui-types/tabs
```

## Usage

```ts
import type { TabsOrientation, TabsActivationMode } from '@sprocketui-types/tabs';

const orientation: TabsOrientation = 'horizontal';
const activation: TabsActivationMode = 'automatic';
```
